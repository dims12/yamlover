# 72-images — EXIF/GPS sample gallery

Test fixtures for yamlover's image renderers and EXIF/GPS extraction. 15 JPEGs
in four themed folders (Paris / Luxembourg). Every image carries the common
camera EXIF — `Make`/`Model` (Google Nexus S), `DateTimeOriginal`,
`ExposureTime`, `FNumber`, `ISO`, `FocalLength`, `Orientation` — and 14 of 15
also carry GPS coordinates.

## Source

Greenstone tutorial sample files (`images_gps.zip`), New Zealand Digital Library
Project — https://files.greenstone.org/tutorial/gs3-current/en/images_gps.htm

Vendored here unmodified as test data. No explicit redistribution license is
stated upstream; these are kept solely as EXIF/GPS fixtures. Replace if a
clearer-licensed set is needed.
